# Secure Login UI

A polished front-end authentication demo built with HTML, CSS, and JavaScript. The project includes animated sign in/sign up panels, form validation, password visibility toggle, password strength feedback, dark/light mode, remember me, and a small dashboard after login.

## Features

- Animated sign in and sign up experience
- Responsive mobile layout
- Dark and light theme with saved preference
- Password strength indicator
- Remember me email persistence
- Demo account storage with `localStorage`
- Dashboard view after successful login
- Logout flow
- Forgot password demo modal
- Clear demo account action
- Dashboard entrance animation

## Demo Notes

This is a portfolio/demo project. User data is stored in `localStorage` so the flow can be tested without a backend. For a real production login system, passwords should be handled on a server and stored only as secure hashes.

## How To Run

Open `index.html` in a browser.
